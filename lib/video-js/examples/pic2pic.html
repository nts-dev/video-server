<script>
	var player = videojs('player_one',{},function(){ 
		this.nuevoPlugin({ 
			logotitle:'Nuevo plugin for Videojs',
			logo: 'images/logo.png',
			logoposition:'RT',
			logourl:'https://www.nuevolab.com'
		}); 
		this.pic2pic({});
	});
</script>

Possible options

  • closeallow - set false to hide close button
  • mute - set true to mute minimized player
this.pic2pic({
	closeallow: false,
	mute: true
});
    Plugin works both: desktop and mobile device, is especially useful on website page with lot of content.
