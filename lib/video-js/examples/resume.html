Playback Resume

Watch part of video then refresh page in your browser. Start resumed playback.

Code Setup

<!--Initialize player and plugin-->
<script>
	var player = videojs('player_one',{},function(){ 
		this.nuevoPlugin({ 
			logotitle:'Nuevo plugin for Videojs',
			logo: 'images/logo.png',
			logoposition:'RT',
			logourl:'https://www.nuevolab.com',
			video_id: 'unique Id',
			resume: true
			
		}); 
	});
</script>

Nuevo plugin includes option to resume playback of a video at the point in time it was left.
You need to enable (true) resume option and set some unique video identifier for option video_id. See example code above.
