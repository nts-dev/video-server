Youtube playback

Code Source

<!--On page youtube video setup-->
<video id="player_one" class="video-js vjs-fluid vjs-default-skin" controls>
	<source src="https://www.youtube.com/watch?v=6UurUzhxs10" type="video/youtube">
</video>

<!--Initialize player and plugin-->
<script type="text/javascript" src="../video.min.js"></script>
<script type="text/javascript" src="../nuevo.min.js"></script>
<script>
	var player = videojs('player_one',nuevo,function(){ 
		this.nuevoPlugin({ 
			logotitle:'Youtube plugin for Videojs',
			logo: 'images/logo.png',
			logoposition:'RT',
			logourl:'https://www.nuevolab.com'
		}); 
	});
</script>
You can use option youtuberesolution:true to show youtube quality levels, however since some time switching between video qualities has been disabled for youtube API player by Youtube and you can't do anything about it.
