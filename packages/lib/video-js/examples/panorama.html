VR360 Panorama

To watch example VR360 panorama video you need to download video files first and upload to "/examples/video" directory.
RIGHT CLICK TO DOWNLOAD
and save as "360dg.mp4"

Use your mouse or fingers on touchscreen to move around 360 degrees in video.Use mouse wheel to zoom in video.

videojs-panorama uses canvas and to avoid a cross domain issue, please keep the code and video within same domain.
For details, please see: For details, please see: https://forums.developer.apple.com/thread/36725

Code Setup

<!--Head tag CSS-->
<link href="../video-js.min.css" rel="stylesheet" type="text/css" />
<link href=../plugins/videojs-panorama.min.css" rel="stylesheet" type="text/css" />


<!--On page above video  tag-->
<script type="text/javascript" src="../video.min.js"></script>
<script type="text/javascript" src="https://cdnjs.cloudflare.com/ajax/libs/three.js/r76/three.js"></script>
<script type="text/javascript" src="../nuevo.min.js"></script>
<script type="text/javascript" src="../plugins/videojs-panorama.min.js"></script>

<!--On page player video tag-->
<video id="player_one" class="video-js vjs-fluid vjs-default-skin" controls preload="auto" poster="images/360dg.jpg" video.crossOrigin="anonymous">
	<source src="video/360dg.mp4" type="video/mp4">
</video>

<!--On page below video tag-->
<script>
	var player = videojs('player_one',{},function(){ 
		this.nuevoPlugin({ 
			logotitle:'VR 360 Panorama Plugin',
			logo: 'images/logo.png',
			logoposition:'RT',
			logourl:'https://www.nuevolab.com'
		}); 
		this.panorama({
			clickAndDrag: true
		});
	});
</script>

For more advanced options check videojs-panorama project on Github page
