Autoplay example

NEVER ASSUME AUTOPLAY WILL WORK.
USING MUTE ATTRIBUTE IMPROVES THE CHANCE THAT AUTOPLAY WILL SUCCEED
  • Apple has updated their autoplay policies to prevent autoplay with sound since Safari 11 release.
  • Chrome's autoplay policies was changed in April of 2018 and works as described in this linked article.
  • Firefox allow users to disable autoplay with a few configuration changes.
If you still wish to test autoplay, rather make use of videojs autoplay option then set autoplay attribute for video element,

Example Player Code Setup

<!--Initialize player and plugin-->
<script>
	var player = videojs('player_one',{autoplay: 'play'},function(){ 
		this.nuevoPlugin({ 
			logotitle: 'Nuevo plugin for Videojs',
			logo: 'images/logo.png',
			logoposition: 'RT',
			logourl: 'https://www.nuevolab.com'
		}); 
	});
</script>
Version 7 of Videojs option autoplay: 'play' call play() on video loadstart instead of using the autoplay attribute.
You may also consider option autoplay: 'any' which also call play() on video loadstart but if it fails, it will try and mute the player and try play()ing again.
